แชท

กร๊วก
ค่าเริ่มต้น

คุยกับกร๊วกเป็นภาษาไทย แล้วมีข้อมูลหน้างานกองอยู่ในหน้าเดียวกัน

  • แผนวันนี้แบบจัดให้
  • ประกาศเตือนภัยอากาศจาก JMA
  • กล้องสด ดูบรรยากาศจริงตอนนี้
  • คำแนะนำแยกหมวด — กิน เที่ยว นอน
  • “กร๊วกเลือกให้ที่นึง” — กดเก็บลงทริปได้

แท็บ

มี 3 แท็บ ไม่ใช่ 4 — app-shell.tsx ประกาศแค่ chat | map | trip เกมฝาท่อถูกถอดออกไปแล้วใน commit a833e48

ทริป

ยังเป็น localStorage เป็นหลัก — อ่านในเครื่องก่อน แล้วค่อย merge จาก server ถ้าตั้ง Supabase ไว้ ถ้าไม่ได้ตั้งก็เงียบๆ ใช้ในเครื่อง UI บอก user ตรงๆ

LLM

มี 3 provider ไม่ใช่หนึ่ง — Anthropic เป็นตัวหลัก มี OpenAI กับ Gemini สำรอง โค้ดเลือกตามคีย์ที่มีใน env

ถอย

มี kill switch — ?shell=0 คืนหน้า dashboard เดิมล้วนแบบไม่มีแท็บ ตั้งใจไว้ถอยหนึ่ง release ถ้าเจอบั๊ก

เทส

ยังไม่มี test runner — ไม่มี vitest/jest ใน devDependencies, script มีแค่ dev / build / start / lint

อ่านจากโค้ดจริงในรีโป — package.json, app/api/*, components/*, lib/services/* ไม่ได้เดาจากความจำ
