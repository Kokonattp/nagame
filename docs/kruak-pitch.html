ทำไมต้องกร๊วก

สิ่งที่ ChatGPT กับ Agoda ให้ไม่ได้

เพื่อน ไม่ใช่เซลส์

กร๊วกแนะนำตามความเหมาะ ไม่ใช่ตามค่าคอม — บางทีแนะศาลเจ้าฟรี ทริคคอนบินิ ที่ไม่ได้เงินสักบาท ความไม่ขายของคือเหตุผลที่คนเชื่อ

บริบทคนไทยจริง

งบเป็นบาท · ฮาลาล/มังสวิรัติ/แพ้กุ้ง · JR Pass คุ้มไหม · คำถามที่คนไทยถามจริงแต่ ChatGPT ตอบแบบฝรั่ง Agoda ไม่ตอบเลย

จำคุณได้ + ทักก่อน

ล็อกอินด้วย LINE — กร๊วกจำทริปที่แล้ว จำว่าแพ้อะไร และทักมาเองตอนใกล้บิน "อาทิตย์หน้าโตเกียวฝนนะ เตรียมร่ม" — โครงสร้างที่คู่แข่งเลียนแบบไม่ได้

Business Model

ฟรีทั้งหมด — รายได้ตามมาเมื่อคนรักมันแล้ว

วันนี้: ฟรี + วัดทุกคลิก

เราไม่ใช่ตัวจอง ไม่รับเงินสักบาท — ทุกปุ่ม "เช็คห้องว่าง" ออกไปเว็บจองผ่านท่อกลางที่นับคลิกไว้ตั้งแต่วันแรก = รู้มูลค่าล่วงหน้าก่อนเปิดรายได้

พรุ่งนี้: เสียบ affiliate

ท่อเดิม เติม affiliate id — ที่พัก (Agoda ~4-7%) · กิจกรรม/JR Pass (Klook ~5-8%) · ตั๋วบิน ไม่ต้องรื้ออะไรเลย และกร๊วกยังแนะนำตามความเหมาะเหมือนเดิม

ถัดไป: เศรษฐกิจความรัก

ซื้อขนม/ชุดให้กร๊วก แบบเดียวกับที่คนซื้อสติกเกอร์ LINE — จ่ายเพราะรัก ไม่ใช่เพราะถูกขาย · ต้นทุน AI เพียง ~$0.003/คำตอบ ถูกกว่าที่คิดมาก

Roadmap

ของโชว์ได้ทุก 1–2 สัปดาห์ ไม่หายไปเป็นเดือน

Phase 0 — แชทเป็นหน้าแรก + การ์ดใบแรก ~1 สัปดาห์

เปิดมาถามกร๊วกได้เลยไม่ต้องเลือกเมือง · คำตอบเป็นการ์ด · ท่อนับคลิกวางตั้งแต่วันแรก

Phase 1 — ข้อมูลจริง 3 แหล่ง ~2-3 สัปดาห์

ที่พัก Rakuten API จริง · ร้าน Google Places (+filter ฮาลาล/แพ้กุ้ง) · ตั๋วบินแบบซื่อสัตย์ (เช็คไม่ได้ก็บอกตรง ๆ)

Phase 2 — กร๊วกร่างทริป + ลากวาง + LINE Login ~2-3 สัปดาห์

"ไป 5 วัน" → ได้ร่างทั้งทริป ลากปรับได้ · ล็อกอิน LINE = กร๊วกจำคุณได้จริง + เริ่มวัด retention จริง

Phase 3 — เลี้ยงกร๊วก retention ระหว่างปี

ให้ขนม/ของฝากรายวัน ปลดร่างใหม่ — เหตุผลเปิดแอปแม้ยังไม่มีทริป (คนเที่ยวญี่ปุ่นปีละครั้ง แต่คิดถึงแมวได้ทุกวัน)

อนาคต — เมืองจำลอง diorama + เมืองที่สอง

เมื่อของหลักพิสูจน์แล้วว่าคนใช้จริง ค่อยเติมโลก 2.5D ที่กร๊วกเดินนำเที่ยว (แผน PixiJS พร้อมอยู่แล้ว)

สิ่งเดียวที่ต้องพิสูจน์

"คนไทยจะเปิดกร๊วกครั้งแรกทำไม"

เทคโนโลยีพร้อม ต้นทุนคุมได้ โมเดลรายได้วางท่อไว้แล้ว — เดิมพันจริงอยู่ที่ hook ประโยคเดียว: "ลองถามแมวตัวนี้ดู" ซึ่งทดสอบได้ถูก ๆ ผ่าน TikTok/เพจ/LINE OA คู่ขนานกับการ build ตั้งแต่ Phase 0
